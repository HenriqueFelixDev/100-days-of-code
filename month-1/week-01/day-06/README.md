## Day 06 - Next Prime

Given an integer, create a function that returns the next prime. If the number is prime, return the number itself.

### Examples

```text
NextPrime(12) ➞ 13

NextPrime(24) ➞ 29

NextPrime(11) ➞ 11
// 11 is a prime, so we return the number itself.
```
